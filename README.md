Chroma Manifest
===================

To initialize your local repository use

    repo init -u https://github.com/whdgmawkd/chroma_manifest.git -b mm6.0
    

Then to sync up:

    repo sync


Build command is

    ./build.sh device -j#


Supported Devices

   Nexus 4  - mako

   Nexus 5  - hammerhead

   Nexus 5X - bullhead

   Nexus 6  - shamu

   Nexus 6P - angler
